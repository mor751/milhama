**Project Setup**

This project contains everything you need to run the app locally.

**Prerequisites:**

1. Install dependencies: `npm install`
2. Run the dev server: `npm run dev`

**Dev Server**

Default URL: `http://localhost:5173` (Vite’s default; if busy it will pick the next available port).

**Build**

`npm run build`

**Preview**

`npm run preview`
